# API Detecção de Fraudes


### Python3 environment
```bash
python3 -m venv venv
source ./venv/bin/activate
python3 -m pip install --upgrade pip
python3 -m pip install -r requirements.txt
```


### Como rodar localmente
- Iniciar Dash API
```bash
python3 src/app.py
```

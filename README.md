# API Detecção de Fraudes


### Python3 environment
```bash
python3 -m venv venv
source ./venv/bin/activate
python3 -m pip install -r requirements.txt
```


### Como rodar localmente
- Iniciar Flask API
```bash
flask --app src/app run
```
